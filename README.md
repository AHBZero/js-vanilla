# Instant Movie Search

This is a simple, responsive, and elegant frontend application for searching movies and TV shows using the OMDb API. It's built with vanilla JavaScript, HTML, and CSS, focusing on a clean user experience and performance.

## Features

- **Instant Search:** Results appear as you type (debounced for performance).
- **Responsive Design:** The layout adapts to different screen sizes, from mobile to desktop.
- **Detailed View:** Click on a movie to see more details in a tooltip or modal view for mobile.
- **Loading & Error States:** Clear feedback is provided to the user during API requests and in case of errors.
- **No Frameworks:** Built with pure, vanilla JavaScript.

## How to Run

1.  Clone this repository.
2.  Open the `index.html` file in your web browser.
3.  Or you can use `python -m http.server 8080 --bind 0.0.0.0` to run a simple local server.

## Technologies Used

-   HTML5
-   CSS3
-   Vanilla JavaScript (ES6+)
